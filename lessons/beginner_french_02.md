# Lesson 2: At the Café

## Learning Objectives
- Order drinks and simple snacks at a French café
- Ask for something politely using "Je voudrais..."
- Ask for the bill
- Understand common café responses

## Steps
- [ ] Step 1: Learn drinks: le café (coffee), le thé (tea), l'eau (water), le jus d'orange (orange juice), un coca (a coke)
- [ ] Step 2: Learn to order: "Je voudrais un café, s'il vous plaît" (I would like a coffee, please)
- [ ] Step 3: Learn snacks: un croissant, une tartine (toast), un gâteau (a cake), un sandwich
- [ ] Step 4: Learn to ask for the bill: "L'addition, s'il vous plaît" and ask the price: "C'est combien?"
- [ ] Step 5: Learn useful café phrases: "C'est pour ici" (it's to stay), "C'est à emporter" (it's to take away), "Avec du lait" (with milk), "Sans sucre" (without sugar)
- [ ] Step 6: Role-play — walk into a café, greet the server, order a drink and a snack, ask for the bill, and say goodbye

## Review from previous lessons
- Always start with "Bonjour" when entering a café
- Use "S'il vous plaît" when ordering and "Merci" when receiving
