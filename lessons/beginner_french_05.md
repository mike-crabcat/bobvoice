# Lesson 5: At the Hotel

## Learning Objectives
- Check in and check out of a hotel
- Ask about your room and hotel facilities
- Report simple problems
- Ask about breakfast and wifi

## Steps
- [ ] Step 1: Learn check-in phrases: "J'ai une réservation" (I have a reservation), "C'est sous le nom de..." (It's under the name of...)
- [ ] Step 2: Learn room vocabulary: la chambre (room), la clé (key), le petit-déjeuner (breakfast), le wifi, le mot de passe (password)
- [ ] Step 3: Learn to ask about facilities: "À quelle heure est le petit-déjeuner?" (What time is breakfast?), "Il y a le wifi?" (Is there wifi?), "Le mot de passe du wifi, s'il vous plaît"
- [ ] Step 4: Learn to report problems: "Il n'y a pas d'eau chaude" (There's no hot water), "La clé ne marche pas" (The key doesn't work), "C'est trop bruyant" (It's too noisy)
- [ ] Step 5: Learn checkout: "Je voudrais régler" (I'd like to check out), "Je pars demain" (I'm leaving tomorrow)
- [ ] Step 6: Role-play — arrive at a hotel, check in with your name, ask about breakfast and wifi, report that the key doesn't work

## Review from previous lessons
- Use "Bonjour" when approaching the reception desk
- Use "S'il vous plaît" and "Merci" throughout
- Use "Excusez-moi" when reporting a problem

## Quiz
- [ ] Q1: Check in at a hotel saying you have a reservation
- [ ] Q2: Ask about breakfast time and the wifi password
- [ ] Q3: Report a problem and say you'd like to check out
