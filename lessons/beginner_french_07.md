# Lesson 7: Days, Times, and Tickets

## Learning Objectives
- Say the days of the week
- Ask about opening and closing times
- Buy tickets for attractions and events
- Understand simple schedules

## Steps
- [ ] Step 1: Learn the days: lundi (Monday), mardi (Tuesday), mercredi (Wednesday), jeudi (Thursday), vendredi (Friday), samedi (Saturday), dimanche (Sunday)
- [ ] Step 2: Learn to ask about times: "À quelle heure?" (At what time?), "C'est ouvert?" (Is it open?), "C'est fermé quand?" (When is it closed?)
- [ ] Step 3: Learn time phrases: le matin (morning), l'après-midi (afternoon), le soir (evening), aujourd'hui (today), demain (tomorrow), maintenant (now)
- [ ] Step 4: Learn ticket phrases: "Un billet, s'il vous plaît" (A ticket please), "Pour aujourd'hui" (For today), "C'est plein?" (Is it full?), "Combien de temps?" (How long does it take?)
- [ ] Step 5: Learn useful attraction phrases: "Où est l'entrée?" (Where is the entrance?), "Où est la sortie?" (Where is the exit?), "Il y a une file d'attente?" (Is there a queue?)
- [ ] Step 6: Role-play — find out what time a museum opens, buy a ticket, ask how long the visit takes, and ask where the exit is

## Review from previous lessons
- Use "Excusez-moi" before asking staff a question
- Use "Combien" for asking about prices and quantities
- Use direction words from Lesson 4 for finding entrance/exit

## Quiz
- [ ] Q1: Ask what day the museum is closed
- [ ] Q2: Buy a ticket and ask what time the attraction closes today
- [ ] Q3: Ask how long something takes and where the exit is
