# Lesson 3: At the Restaurant

## Learning Objectives
- Order a full meal at a French restaurant
- Ask what something is on the menu
- Express dietary needs and preferences
- Ask for the menu and understand the courses

## Steps
- [ ] Step 1: Learn to ask for a table: "Une table pour deux, s'il vous plaît" (a table for two) and "Le menu, s'il vous plaît"
- [ ] Step 2: Learn meal courses: l'entrée (starter), le plat principal (main course), le dessert, les boissons (drinks)
- [ ] Step 3: Learn to ask about the menu: "Qu'est-ce que c'est?" (What is it?), "Qu'est-ce que vous recommandez?" (What do you recommend?)
- [ ] Step 4: Learn main dish vocabulary: le poulet (chicken), le bœuf (beef), le poisson (fish), les pâtes (pasta), la salade (salad), les frites (fries)
- [ ] Step 5: Learn dietary phrases: "Je suis végétarien" (I'm vegetarian), "Je n'aime pas..." (I don't like...), "Une autre chose" (something else)
- [ ] Step 6: Role-play — enter a restaurant, get a table, ask about the menu, order a starter and main course, ask for the bill

## Review from previous lessons
- Use "Bonjour" when greeting the waiter
- Use "Je voudrais" from Lesson 2 for ordering
- Use "Merci" and "S'il vous plaît" throughout

## Quiz
- [ ] Q1: Ask for a table and the menu in French
- [ ] Q2: Ask "What do you recommend?" then order a main course
- [ ] Q3: Say you are vegetarian and ask for something else
